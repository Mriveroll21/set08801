Learn About Ciphers

Caesar Cipher

The Caesar cipher is a substitution cipher that replaces each letter in a text by a letter some fixed number of positions down the alphabet.

Example:
HELLO → KHOOR

Reverse Cipher

The Reverse cipher is a simple encryption method that flips the entire message backwards, making it unreadable at a glance.

Example:
HELLO → OLLEH
